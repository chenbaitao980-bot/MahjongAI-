### 二、未实现项详细说明

#### 05 轻量蒙特卡洛（完全缺失）
- 缺少文件：`game/monte_carlo.py`, `game/rollout.py`
- 缺失功能：
  - `build_unknown_tiles()` — 从完整牌池扣除所有可见牌
  - `sample_hidden_state()` — 随机分配未知牌给对手
  - `simple_rollout_policy()` — 模拟中使用的简单AI
  - `rollout_until_end()` — 模拟对局直到结束
  - `evaluate_action_mc()` — 对单个候选动作模拟N次

- 当前系统仅依赖本地向听数/进张/危险度评分，无模拟验证

#### 06 大模型解释与最终输出（部分缺失）
- 已有：DeepSeek API 调用、Prompt 模板、JSON 输出解析
- 缺失：
  - `validate_llm_output()` — 检查大模型输出是否在 `legal_actions` 中

  - `get_program_advice()` — 无大模型模式下的纯程序建议
  - `get_final_advice()` — 总入口（state → analyze → MC → LLM → validate → final）
  - Prompt 中要求 LLM "不能自行重新计算"，但无技术层面的校验拦截
